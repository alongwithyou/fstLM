# fstLM
WFST Format Language Model

# Perplexity

fstLM can compute perplexity for a language model converted
into WFST format.

## Usage
### Build

Run following commands to build this project:

```shell
$ git clone https://github.com/wantee/fstLM.git
$ cd fstLM/src
$ make
```

Quick running:

```shell
$ cd $FSTLM_HOME/egs/tedlium
$ ./run.sh
```
